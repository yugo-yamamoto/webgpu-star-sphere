# WebGPU Star Sphere

透明な球体の内部にオレンジ色の星状パーティクルが浮かび、ゆっくりと回転するアニメーション。

## デモ

<video src="2026-04-14_000254.mp4" controls autoplay loop muted playsinline></video>

`star-sphere.html` をブラウザで開くだけで動作します。

## 動作環境

- Chrome 113 以降
- Edge 113 以降
- その他 WebGPU 対応ブラウザ

## 仕様

| 項目 | 内容 |
|---|---|
| パーティクル数 | 1,500個 |
| 配置 | 球体内部（一様分布）＋表面付近シェル |
| 色 | オレンジ（中心は白熱） |
| 背景 | 黒 |
| 回転 | Y軸周りに低速回転 |
| サイズ | ウィンドウに合わせて最大化（アスペクト比維持） |

## 技術

- **WebGPU** — レンダリングパイプライン、ストレージバッファ、インスタンス描画
- パーティクルをビルボードクワッドとして描画（6頂点 × 1,500インスタンス）
- ガウシアン減衰＋加算ブレンディングによるグロー表現
- 直交投影、Y軸回転行列をユニフォームバッファで毎フレーム更新
